# Telecom Customer Churn Prediction

A Machine Learning classification project that predicts whether a telecom customer is likely to churn based on customer demographics, account information, and service usage.

---

## Project Overview

Customer churn is one of the biggest challenges faced by telecom companies. This project aims to build multiple machine learning models to identify customers who are likely to leave the company.

The project follows an industry-standard machine learning workflow including data preprocessing, feature engineering, model training, hyperparameter tuning, and performance evaluation.

---

## Dataset

- Telecom Customer Churn Dataset
- Target Variable: `Churn`
  - 0 → Customer Stays
  - 1 → Customer Churns
